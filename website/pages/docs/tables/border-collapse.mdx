# Border Collapse

Utilities for controlling whether table borders should collapse or be separated.

| React props                | CSS Properties                |
| -------------------------- | ----------------------------- |
| `borderCollapse={keyword}` | `border-collapse: {keyword};` |

## Responsive

To control the border collapsing of an element at a specific breakpoint, use responsive object notation. For example, adding the property `borderCollapse={{ md: "collapse" }}` to an element would apply the `borderCollapse="collapse"` utility at medium screen sizes and above.

```jsx
<x.table borderCollapse={{ md: 'collapse' }} />
```
